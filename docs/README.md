# Math formulas
## Area
- Circle: S = πR²
```
R = 5 
Circle area = 78,5398...  
```
- Rectangle: S = ab
```
a = 5, b = 3
Rectangle area = 15
```
- Square: S = a²
```
a = 6
Square area = 36
```
- Triangle: S = a * h / 2
```
a = 4, b = 7
Triangle area = 14
```

## Perimeter
- Circle: P = 2πR
```
R = 4
Circle perimetr = 25,1327...
```
- Rectangle: P = 2a + 2b
```
a = 3, b = 8
Rectangle perimetr = 22
```
- Square: P = 4a
```
a = 6
Square perimetr = 24
```
- Triangle: P = a + b + c

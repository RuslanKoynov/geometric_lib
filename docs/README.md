# Math formulas
A library with formulas for areas and perimeters of simple geometric shapes.

## Area
- Circle: S = πR²
```
R = 2
S = circle.area(R)
```
- Rectangle: S = ab
```
a = 3
b = 4
S = rectangle.area(a, b)
```
- Square: S = a²
```
a = 5
S = square.area(a)
```
- Triangle: S = ah/2
```
a = 7
h = 8
S = triangle.area(a, h)
```

## Perimeter
- Circle: P = 2πR
```
R = 2
S = circle.perimeter(R)
```
- Rectangle: P = 2a + 2b
```
a = 3
b = 4
S = rectangle.perimeter(a, b)
```
- Square: P = 4a
```
a = 5
S = square.perimeter(R)
```
 - Triangle: P = a + b + c
```
a = 3
b = 4
c = 5
S = triangle.perimeter(a, b, c)
```
